# Shared Documentation Assets

This directory stores assets that can be reused by multiple language versions.

## Rules

- Put language-neutral images in `assets/`.
- Put language-neutral diagrams in `diagrams/`.
- Keep prose documents out of `_shared/`.
- Put localized screenshots in the matching language directory.
- Prefer stable file names that describe the subject, such as `device-link-frame.png`.
