# 术语表

## Terms

| Term | Translation | Meaning | Preferred Usage | Avoid |
| --- | --- | --- | --- | --- |
| firmware | 固件 | 运行在 MCU 或嵌入式设备侧的软件。 | 用于设备侧软件。 | 不用于普通 PC 应用。 |
| bring-up | 板级启动 | 让新硬件进入可启动、可下载、可调试状态。 | 用于新板调试。 | 不简单翻译成启动。 |
| driver | 驱动 | 控制硬件外设或操作系统设备的代码层。 | 区分 MCU 驱动和 Linux 驱动。 | 不泛指所有模块。 |
| protocol | 协议 | 设备、主机或服务之间交换数据的规则。 | 用于帧格式、命令和载荷约定。 | 不和 API 混用。 |
| OTA | 空中升级 | 通过通信链路更新固件或软件镜像。 | 保留 OTA 缩写。 | 不省略校验和回滚语义。 |
| RTOS | 实时操作系统 | 面向确定性调度和嵌入式任务管理的操作系统。 | 保留 RTOS 缩写。 | 不等同于通用 Linux。 |
| task | 任务 | RTOS 中的执行单元。 | 用于 FreeRTOS 等上下文。 | 在 Zephyr 语境中优先使用 thread。 |
| IPC | 进程间通信 | 执行单元之间传递数据或事件的机制。 | 可用于任务、线程或进程间通信。 | 不只限定为进程。 |
| device tree | 设备树 | Linux 中描述硬件结构的数据。 | 保留英文术语并给出中文解释。 | 不翻译成设备树文件时忽略 binding。 |
| kernel module | 内核模块 | 可加载到 Linux 内核中的功能模块。 | 用于驱动和内核扩展。 | 不和用户态服务混用。 |
| ROS2 node | ROS2 节点 | ROS2 系统中的基本运行单元。 | 首次出现写 ROS2 节点。 | 不省略 ROS2 导致歧义。 |
| topic | 话题 | ROS2 中的发布订阅数据通道。 | 用于数据流。 | 不用于请求响应语义。 |
| service | 服务 | ROS2 中的请求响应接口。 | 用于同步请求响应。 | 不用于连续数据流。 |
| action | 动作 | ROS2 中可反馈、可取消的长任务接口。 | 用于导航等长耗时任务。 | 不和 service 混用。 |
| simulation | 仿真 | 在软件环境中模拟机器人、传感器或环境。 | 用于 Gazebo、RViz 相关内容。 | 不等同于单元测试。 |
| control loop | 控制环 | 周期性读取状态、计算控制量并输出执行命令的闭环。 | 用于电机和机器人控制。 | 不和普通业务循环混用。 |
| sensor fusion | 传感器融合 | 组合多个传感器以估计更可靠状态。 | 用于 IMU、编码器、里程计等组合。 | 不等同于简单数据拼接。 |
| edge AI | 边缘 AI | 在本地设备或边缘计算平台上运行 AI 推理。 | 用于 Jetson、NPU、端侧推理。 | 不泛指云端 AI。 |
| agent | 智能体 | 能根据目标、工具和上下文进行计划或行动的软件实体。 | 用于 AI 控制和工具调用边界。 | 不用于普通脚本。 |
| embodied AI | 具身智能 | 与物理系统、感知和行动闭环连接的 AI。 | 用于机器人和真实环境交互。 | 不等同于聊天机器人。 |
